# AppLabJS

A JavaScript Library allow to use functions in code.org App Lab in browser enviroment

# Installation

Add this script tag to your HTML code

```` html
<script src="https://cdn.jsdelivr.net/gh/richardliucode/applabjs@1.0.1/applab.min.js"></script>
````

# Usage
- You can see the document of [code.org App Lab](https://studio.code.org/docs/ide/applab/), it use the similar api
